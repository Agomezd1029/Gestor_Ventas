# Gestor de Ventas

Este proyecto implementa un sistema básico para gestionar ventas en una tienda.  
Permite **registrar productos, realizar ventas, consultar historial y generar reportes**, almacenando la información en **CSV** y **JSON**.

## 🚀 Funcionalidades
- Menú interactivo con opciones CRUD.
- Registro de productos (ID, nombre, precio, stock).
- Registro de ventas (producto, cantidad, total).
- Consulta de productos y ventas.
- Modificación y eliminación de productos.
- Persistencia de datos en archivos CSV y JSON.
- Manejo de excepciones para evitar errores comunes.

## 📚 Temas aplicados
- **Fundamentos de programación**: variables, condicionales, bucles.
- **Funciones**: modularización del código.
- **Colecciones**: listas y diccionarios.
- **Archivos**: lectura/escritura en CSV y JSON.
- **Excepciones**: manejo de errores.
- **CRUD**: operaciones completas sobre productos y ventas.
